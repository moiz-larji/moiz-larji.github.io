# Moiz Larji Portfolio

Personal portfolio website hosted on GitHub Pages.

## Stack
- HTML
- CSS
- Vanilla JavaScript

## Run locally
Open `index.html` in your browser.

## Update content
- Main structure/content: `index.html`
- Styling: `src/style.css`
- Interactions/animations: `src/app.js`

## Deploy
Push to `main` branch. GitHub Pages serves the site at:
`https://moiz-larji.github.io/`
